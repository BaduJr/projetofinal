# Configuração

1) Clone ou baixe este repositório (https://github.com/ederfranco23/academy-store-api) e, estando no diretório do projeto, abra o console: 
`npm install`

2) Para executar, estando no diretório do projeto, abra o console: 
`npm start`

3) Clone ou baixe este repositório (https://github.com/BaduJr/projetofinal) e, estando no diretório do projeto, abra o console: 
`npm install`

4) Para executar, estando no diretório do projeto, abra o console: `ng serve` e acesse `http://localhost:4200` 

---

# Utilização

1) Na barra superior de navegação, haverá duas opções de menu: Home e Produtos

2) O menu home é a página inicial do projeto. Que contém um título

3) O menu produtos é a página de listagem dos produtos. Quem contém a listagem dos produtos

4) Para criar um novo produto, clique no botão Novo e preencha o formulário que se apresentará na tela e clique em Salvar

5) Para alterar um produto, clique no botão Alterar e altere os dados que se encontram no formulário e clique em Alterar

6) Para excluir um produto, clique no botão Deletar
